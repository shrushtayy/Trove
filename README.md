# Trove 💚

A Gen Z lifestyle finance app that helps you afford the life you actually want — concerts, cafés, travel — without guilt trips or boring budgeting.

🔗 **Live Demo:** https://trove-jet-ten.vercel.app

## What Trove does
- Shows your real spendable money (vibe budget)
- Helps you save for concerts, trips, and experiences
- Finds subscriptions silently draining you
- Splits bills via UPI
- AI math buddy — "Can I afford this?"
- Credit card coach and CIBIL explainer

## Tech Stack
React Native + Expo · Supabase · Claude API · Vercel
